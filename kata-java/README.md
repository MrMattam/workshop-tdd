# Les besoins

* Afficher les chiffres de 1 Ã  100.

* Si un chiffre est divisible par 3 on affiche Fizz
* Si un chiffre est divisible par 5 on affiche Buzz
* Si un chiffre est multiple de 3 ET de 5 alors, on affiche FizzBuzz

## Exemple de sortie :

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
Buzz
... etc up to 100
```
